Privacy Policy

Last updated: July 2026

Genshin Wish Calculator does not collect, store, or transmit any personal data, and there is no account system of any kind.

Analytics

This site uses Cloudflare Web Analytics to see aggregate traffic trends, like how many people visit and which pages are popular. It's a cookieless, privacy-first analytics service: it doesn't use tracking cookies, doesn't fingerprint your device, and doesn't build a profile of you across sites. It only reports anonymous, aggregate numbers (e.g. total page views) — this site's owner cannot see who visited, what they did, or any other personally identifying information.

What stays on your device

Any information you enter — pity counts, inventory, build data, saved settings — is stored using your browser's localStorage. This data never leaves your device and is never sent to any server. Clearing your browser data or using a different browser/device will remove it.

Third-party data sources

To show up-to-date game data, this site pulls publicly available information from:

  • Ambr.top — character, weapon, and banner data
  • Ennead.cc — event calendar data
  • Hashblen — redeem code data

These requests happen from your browser directly to those services to display data on the page; no personal information is sent as part of them beyond standard, anonymous HTTP request info (which is normal for any web request and outside this site's control).

Hosting

This site is hosted on Cloudflare Pages, which may log basic, anonymized request data (e.g. IP address, timestamp) for infrastructure and security purposes, as is standard for any web host. This site's owner does not have access to personally identifying analytics.

Changes

If this policy ever changes (e.g. if ads are added in the future), this page will be updated to reflect that.
